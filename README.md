# Dirty Cafe Sales Analysis Dashboard

This project focuses on cleaning, analyzing, and visualizing sales data of a cafe to generate meaningful business insights.

## Objective
- To work with real-world dirty sales data
- To analyze sales performance
- To create a clean and interactive dashboard for decision-making

## Key Tasks Performed
- Cleaned raw and inconsistent sales data
- Prepared structured data for analysis
- Created pivot tables for aggregation
- Built an interactive Excel dashboard
- Analyzed:
  - Item-wise sales
  - Monthly sales trends
  - Sales by payment method
  - Sales by location

## Tools Used
- Python(Pandas, Matplotlib) -(for initial data cleaning)
- Microsoft Excel (Pivot Tables, Charts, Dashboard)-(for analysis and dashboard creation)

## Files Included
- Dirty_Cafe_Sales_Analysis_Dashboard.xlsx

## Note
Initial data cleaning was performed using Python. The cleaned dataset was then used in Microsoft Excel to create the final dashboard and visualizations.
## Future Scope
- Advanced analysis and dashboarding using Power BI
- Predictive insights using Machine Learning models

## Outcome
This project provides clear insights into sales patterns and helps understand customer behavior and business performance.
